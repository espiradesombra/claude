Fase D – Reinicio:
El otro módulo hará lo mismo, recibiendo asistencia del primero cuando a él le toque su “momento de baja inercia”.

    cx, cy = center
    # Ángulo del eje (theta_r)
    if f < 24:
        theta_r = 0.0 + 0.01*f
    elif f < 36:
        theta_r = 0.24 + 0.12*(f-24)  # empuje
    elif f < 60:
        theta_r = 0.24 + 0.12*12 + 0.08*(f-36)  # inercia
        # corregir: 0.24+1.44=1.68
