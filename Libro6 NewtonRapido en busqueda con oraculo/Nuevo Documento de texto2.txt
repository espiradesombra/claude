\begin{abstract}
Presentem una demostració, en el marc de la teoria de densitats asimptòtiques, que força la infinitud del conjunt de primers de Sophie Germain $L_{\mathrm{sg}}$ dins l’univers dels enters de la forma $6k-1$ ($L_1$). El nucli està en un balanç de creixements entre $L_3$, $L_4$, la seua intersecció $L_2$, i $L_{\mathrm{sg}}$, amb Principi d’Inclusió--Exclusió (PIE) i factorització del ritme base $|L_1(N)|\sim N/6$. Mostrem que l’hipòtesi “$L_{\mathrm{sg}}$ finit/cte” introdueix una contradicció en les igualtats de creixement, de manera que necessàriament $L_{\mathrm{sg}}(N)\to\infty$.
\end{abstract}

\section{Introducció}
Considerem l’univers $L_1=\{n\in\mathbb{N}: n\equiv -1 \pmod{6}\}$, amb $|L_1(N)|\sim N/6$. Estudiem una descomposició de $L_1$ en conjunts associats a la fenomenologia de Sophie Germain i provem un teorema d’infinitud per contradicció asimptòtica via balanç de densitats i PIE.

\section{Conjunts i notació}
\begin{definition}[Univers i subconjunts]
$L_1=\{6k-1\}$; $L_3$ i $L_4$ són famílies de compostos dins $L_1$; $L_2:=L_3\cap L_4$; $L_{\mathrm{sg}}$ és el conjunt de primers de Sophie Germain en el marc $6k-1$.
\end{definition}

\begin{remark}[Forma equivalent]
$L_1=(L_3\setminus L_4)\,\dot{\cup}\,(L_4\setminus L_3)\,\dot{\cup}\,L_2\,\dot{\cup}\,L_{\mathrm{sg}}$.
\end{remark}

Definim densitat relativa dins $L_1$: $\delta_X(N):=|X(N)|/|L_1(N)|$.

\section{Identitats i PIE}
Per PIE,
\begin{equation}\label{eq:pie}
|L_1(N)|=|L_3(N)|+|L_4(N)|-|L_2(N)|+|L_{\mathrm{sg}}(N)|.
\end{equation}
Dividint \eqref{eq:pie} per $|L_1(N)|$:
\[
1=\delta_{L_3}(N)+\delta_{L_4}(N)-\delta_{L_2}(N)+\delta_{L_{\mathrm{sg}}}(N).
\]

\begin{theorem}[Infinitud de $L_{\mathrm{sg}}$]
Sota $|L_1(N)|\sim N/6$ i la descomposició anterior, si $L_{\mathrm{sg}}$ fóra finit es produiria una contradicció en el balanç de densitats. Per tant, $|L_{\mathrm{sg}}(N)|\to\infty$.
\end{theorem}

\begin{proof}
Suposem, per absurd, que $|L_{\mathrm{sg}}(N)|=O(1)$; aleshores $\delta_{L_{\mathrm{sg}}}(N)\to 0$. Del balanç relatiu s’obtindria
\[
1=\delta_{L_3}+\delta_{L_4}-\delta_{L_2}.
\]
Tanmateix, per l’estructura interna de $L_3,L_4$ i la correcció de doble comptatge $L_2$, les parts compostes no poden saturar tota la massa de $L_1$ sense la contribució no composta de $L_{\mathrm{sg}}$. Això contradiu $1=\delta_{L_3}+\delta_{L_4}-\delta_{L_2}$. Concloem que $L_{\mathrm{sg}}$ no és finit i, en conseqüència, $|L_{\mathrm{sg}}(N)|\to\infty$.
\end{proof}

\section{Observacions i limitacions}
L’argument és asimptòtic (no efectiu), però estableix necessàriament la infinitud de $L_{\mathrm{sg}}$. Refinaments via criba multiplicativa poden aportar cotes inferiors.

# meq_tester_v2.py
import numpy as np
from math import exp, log, sqrt
from dataclasses import dataclass, field
from typing import List, Optional, Dict

@dataclass
class LogObs:
    y: float           # valor central de log(E)
    eps: float         # tolerància ± en log(E)
    tag: str = ""

@dataclass
class MEQTesterV2:
    tau: float = 1e-3          # llindar de col·lapse: sigma_min/sigma_max < tau
    consensus: float = 0.90    # % de bootstraps “degenerats” per declarar ME local/global
    bootstrap_N: int = 600
    seed: int = 2026
    obs: List[LogObs] = field(default_factory=list)

    def add_log_obs(self, y: float, eps: float, tag: str = ""):
        self.obs.append(LogObs(y=y, eps=eps, tag=tag))

    def _features_from_E(self, E: float) -> np.ndarray:
        # Features no trivials (evitem relacions exactes): totes funcionals de E
        y = log(max(E, 1e-300))
        f1 = y
        f2 = log(1.0 + sqrt(E))
        f3 = log(1.0 + E**(1/3))
        f4 = log(1.0 + E**(1/5))
        return np.array([f1, f2, f3, f4], dtype=float)

    def _sample_matrix(self, rng: np.random.Generator, indices: Optional[List[int]] = None) -> np.ndarray:
        rows = []
        src = self.obs if indices is None else [self.obs[i] for i in indices]
        for ob in src:
            y_s = rng.uniform(ob.y - ob.eps, ob.y + ob.eps)
            E_s = exp(y_s)
            rows.append(self._features_from_E(E_s))
        Y = np.vstack(rows)
        # Centrat de columnes (treiem mitjana): prova de col·lapse afí
        Yc = Y - Y.mean(axis=0, keepdims=True)
        return Yc

    def _degenerate_fraction(self, indices: Optional[List[int]] = None) -> float:
        rng = np.random.default_rng(self.seed)
        deg = 0
        for _ in range(self.bootstrap_N):
            Yc = self._sample_matrix(rng, indices)
            if Yc.shape[0] < 3 or Yc.shape[1] < 2:
                continue
            S = np.linalg.svd(Yc, full_matrices=False, compute_uv=False)
            if S[0] <= 0: 
                continue
            if (S[-1] / S[0]) < self.tau:
                deg += 1
        return deg / max(1, self.bootstrap_N)

    def local_tests(self, window: int = 3) -> List[Dict]:
        n = len(self.obs)
        out = []
        if n < window: return out
        for s in range(0, n - window + 1):
            idx = list(range(s, s + window))
            df = self._degenerate_fraction(indices=idx)
            out.append({ "indices": idx, "degenerate_fraction": df, "local_ME_exists": df >= self.consensus })
        return out

    def global_unification_test(self) -> Dict:
        df = self._degenerate_fraction(indices=None)
        return {"degenerate_fraction_all": df, "global_ME_exists": df >= self.consensus}

    def decision(self, window: int = 3) -> Dict:
        L = self.local_tests(window)
        G = self.global_unification_test()
        k = sum(1 for r in L if r["local_ME_exists"]) if L else 0
        return { "local_windows": L, "global": G, "num_local_ME": k,
                 "non_unification": (k > 0) and (not G["global_ME_exists"]) }

def pca_rank_r_reconstruction_error(Yc: np.ndarray, r: int=1) -> float:
    # Yc: dades centrades (x × d). Retorna RMSE normalitzat per norma Frobenius.
    U, S, Vt = np.linalg.svd(Yc, full_matrices=False)
    # Retalla a rang r
    Sr = np.zeros_like(S)
    Sr[:r] = S[:r]
    Y_hat = (U * Sr) @ Vt
    num = np.linalg.norm(Yc - Y_hat, 'fro')
    den = np.linalg.norm(Yc, 'fro') + 1e-15
    return float(num/den)

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
ME locals (quadràtica/cúbica) i test de NO-unificació
Autor: VMA + Copilot
Ús: python me_local_test.py

Què fa:
- Construeix finestres locals al voltant de p^2 (quadràtic) i p^3 (cúbic)
- Fa bootstrap + SVD per detectar col·lapse local (ME local)
- Comprova el col·lapse global (totes les finestres juntes)
- Si locals=Sí i global=No -> NO-UNIFICACIÓ al teu model

Com ampliar:
- Ajusta les llistes PRIMES_SQUARE i PRIMES_CUBE (per ex. afegeix 13,17,19)
- Ajusta EPS_SQ i EPS_CB si vols més/menys tolerància al log
- Ajusta TAU/CONSENSUS/BOOTSTRAP_N segons la finor del test
"""
from dataclasses import dataclass, field
from typing import List, Optional, Dict, Tuple
import numpy as np
from math import log, exp

# ---------- Dades d'entrada (edita ací) ----------
PRIMES_SQUARE = [5, 7, 11]          # finestra quadràtica: 5^2, 7^2, 11^2
PRIMES_CUBE   = [5, 7, 11]          # finestra cúbica:    5^3, 7^3, 11^3
# Exemple d’ampliació (descomenta si vols):
# PRIMES_CUBE   = [5, 7, 11, 13, 17, 19]

EPS_SQ = 0.15   # tolerància (±) en log(E) per a quadrats
EPS_CB = 0.18   # tolerància (±) en log(E) per a cubes

# Paràmetres del test (pots ajustar-los)
TAU          = 5e-3   # llindar col·lapse: sigma_min/sigma_max < TAU => degenerat
CONSENSUS    = 0.90   # % de bootstraps necessaris per declarar ME local/global
BOOTSTRAP_N  = 900    # mostres bootstrap
SEED         = 2026   # llavor RNG


# ---------- Motor de test ----------
@dataclass
class LogObs:
    y: float       # valor central (log(E))
    eps: float     # tolerància ± en log(E)
    tag: str = ""  # etiqueta opcional

@dataclass
class MEQTesterV3:
    tau: float = TAU
    consensus: float = CONSENSUS
    bootstrap_N: int = BOOTSTRAP_N
    seed: int = SEED
    obs: List[LogObs] = field(default_factory=list)

    def add_log_obs(self, y: float, eps: float, tag: str = ""):
        self.obs.append(LogObs(y=y, eps=eps, tag=tag))

    @staticmethod
    def _near_k_power_residual(E: float, k: int) -> float:
        """Distància a la k-èsima potència perfecta més propera (en l’espai real)."""
        x = E ** (1.0 / k)
        r = round(x)
        return abs(x - r)

    def _features_from_E(self, E: float) -> np.ndarray:
        """
        Features per distingir règims quadràtic vs cúbic:
        - f1: log(E)
        - f2: log(1 + distància a quadrat perfecte)
        - f3: log(1 + distància a cub perfecte)
        - f4: log(1 + distància a 5a potència perfecta)
        - f5: contrast entre distàncies a quadrat/cub (qualsevol règim dominant)
        """
        y = log(max(E, 1e-300))
        d2 = self._near_k_power_residual(E, 2)
        d3 = self._near_k_power_residual(E, 3)
        d5 = self._near_k_power_residual(E, 5)
        f1 = y
        f2 = np.log(1.0 + d2)
        f3 = np.log(1.0 + d3)
        f4 = np.log(1.0 + d5)
        mn = min(d2, d3) + 1e-12
        mx = max(d2, d3) + 1e-12
        f5 = np.log(mx / mn)
        return np.array([f1, f2, f3, f4, f5], dtype=float)

    def _sample_matrix(self, rng: np.random.Generator, indices: Optional[List[int]] = None) -> np.ndarray:
        rows = []
        src = self.obs if indices is None else [self.obs[i] for i in indices]
        for ob in src:
            y_s = rng.uniform(ob.y - ob.eps, ob.y + ob.eps)  # bootstrapping dins de la tolerància
            E_s = exp(y_s)
            rows.append(self._features_from_E(E_s))
        Y = np.vstack(rows)
        # centrat de columnes (treiem mitjana): provem col·lapse afí
        return Y - Y.mean(axis=0, keepdims=True)

    def _degenerate_fraction(self, indices: Optional[List[int]] = None) -> float:
        rng = np.random.default_rng(self.seed)
        deg = 0
        for _ in range(self.bootstrap_N):
            Yc = self._sample_matrix(rng, indices)
            if Yc.shape[0] < 3 or Yc.shape[1] < 2:
                continue
            S = np.linalg.svd(Yc, full_matrices=False, compute_uv=False)
            if S[0] <= 0:
                continue
            if (S[-1] / S[0]) < self.tau:
                deg += 1
        return deg / max(1, self.bootstrap_N)

    def local_tests(self, window_indices: List[List[int]]) -> List[Dict]:
        out = []
        for idx in window_indices:
            df = self._degenerate_fraction(indices=idx)
            out.append({
                "indices": idx,
                "degenerate_fraction": df,
                "local_ME_exists": df >= self.consensus
            })
        return out

    def global_unification_test(self) -> Dict:
        df = self._degenerate_fraction(indices=None)
        return {"degenerate_fraction_all": df, "global_ME_exists": df >= self.consensus}


# ---------- Utilitats ----------
def build_power_logs(primes: List[int], k: int, eps: float, label: str) -> List[Tuple[float, float, str]]:
    """
    Construeix (log(E), eps, tag) per a E = p^k sobre primes donades.
    label s’afegeix al tag per separar finestres (p.ex. 'SQ' o 'CB').
    """
    out = []
    for p in primes:
        E = float(p ** k)
        out.append((log(E), eps, f"{int(E)}={p}^{k} [{label}]"))
    return out

def print_human_summary(title: str, local_res: List[Dict], global_res: Dict):
    print("\n" + "="*70)
    print(f"{title}")
    print("="*70)
    for i, r in enumerate(local_res, 1):
        idx = r["indices"]
        print(f"- Finestra {i} (índexos {idx}): "
              f"deg_frac={r['degenerate_fraction']:.3f}  "
              f"ME_local={r['local_ME_exists']}")
    print(f"Global: deg_frac_all={global_res['degenerate_fraction_all']:.3f}  "
          f"ME_global={global_res['global_ME_exists']}")
    # Decisió de NO-unificació (locals sí, global no):
    k = sum(1 for r in local_res if r["local_ME_exists"])
    non_unif = (k > 0) and (not global_res["global_ME_exists"])
    print(f"NO-UNIFICACIÓ (locals sí i global no)?  {non_unif}")
    print("="*70)


# ---------- Demo principal ----------
def main():
    tester = MEQTesterV3()

    # 1) Afegim finestra QUADRÀTICA
    sq_obs = build_power_logs(PRIMES_SQUARE, k=2, eps=EPS_SQ, label="SQ")

    # 2) Afegim finestra CÚBICA

    offset = len(sq_obs)
    for y, eps, tag in cb_obs:

    # Índexos de finestra (locals)
    sq_idx = list(range(0, len(sq_obs)))                  # [0..len(sq)-1]
    cb_idx = list(range(offset, offset + len(cb_obs)))    # [offset..offset+len(cb)-1]

    local_results = tester.local_tests([sq_idx, cb_idx])

    # 4) Test global (totes les files)
    global_result = tester.global_unification_test()

    # 5) Resum humà
    print_human_summary("ME locals quadràtica i cúbica", local_results, global_result)

    # 6) També escopim JSON estructurat (per si vols parsejar)
    import json
    out = {
        "params": {
            "tau": tester.tau,

            "eps_sq": EPS_SQ,

        },

        "global": global_result,

    }
    print("\nJSON_RESULT =")

if __name__ == "__main__":

def tsp_local_ME(points, X_min=2, X_max=6, T=5, kNN=20):
    n = len(points)
    # Inicialitza recorregut (Greedy NN o MST -> preorder, etc.)
    tour = initial_greedy_tour(points)

    # Precalcula veïnats per reduir triples
    Nbh = {i: k_nearest(points, i, kNN) for i in range(n)}

    for t in range(T):  # T rondes macro
        X = schedule(t, X_min, X_max)  # dimensió de l'exploració

        # 1) Construir llista de triples i "scores d'estalvi"
        triple_list = []
        for u in range(n):
            for v in Nbh[u]:
                for w in Nbh[u]:
                    if v == w: 
                        continue
                    S = saving_score(points, u, v, w)  # la teua fórmula
                    D = d(points[u], points[v]) + d(points[u], points[w])  # o un proxy de "compactesa"
                    triple_list.append((S, D, u, v, w))

        # 2) Ordenar: 1r per S desc, 2n per D asc
        triple_list.sort(key=lambda x: (-x[0], x[1]))

        improved = False
        # 3) Explorar "X" millors triples per cada u (control de dimensió)
        for (S, D, u, v, w) in head_by_u(triple_list, X):
            # 3.1) Generar moviments locals al tour (2-opt, 3-opt, o insercions condicionades per (u,v,w))
            candidates = generate_moves(tour, u, v, w, X)

# -*- coding: utf-8 -*-
"""
Heurística geomètrica per al TSP amb ME locals i dimensió X
-----------------------------------------------------------

- Score d'estalvi per triples (u,v,w)
- Ordenació per score desc i distància asc
- Exploració controlada per X (dimensió)
- 2-opt focalitzat al voltant del triple
- Selecció del millor guany heurístic

Autor: VMA + Copilot (2026-01-24)
"""

import math
import random
from collections import defaultdict
from typing import List, Tuple

Point = Tuple[float, float]
Tour = List[int]

# ---------- Distàncies i utilitats ----------

def dist(p: Point, q: Point) -> float:
    return math.hypot(p[0]-q[0], p[1]-q[1])

def total_length(points: List[Point], tour: Tour) -> float:
    L = 0.0
    n = len(tour)
    for i in range(n):
        a = points[tour[i]]
        b = points[tour[(i+1) % n]]
        L += dist(a, b)
    return L

def positions_in_tour(tour: Tour) -> dict:
    """Mapeig node -> posició al tour (O(n))."""
    return {node: i for i, node in enumerate(tour)}

# ---------- Score d'estalvi per triples ----------

def saving_score(points: List[Point], u: int, v: int, w: int, eps: float = 1e-9) -> float:
    """
    Interpretació de la teua fórmula:
    S(u,v,w) = ((d(u,w)+d(u,v))^2 / (d(u,v)+eps)) * d(u,w)
    Després normalitzem lleugerament per evitar dominàncies exagerades.
    """
    duv = dist(points[u], points[v])
    duw = dist(points[u], points[w])
    raw = ((duw + duv) ** 2) / (duv + eps) * duw
    # Normalització suau per no sobrepremiar distàncies molt grans:
    norm = raw / (duw + duv + 1e-9)
    return norm

# ---------- Inicialització (tour greedy) ----------

def initial_greedy_tour(points: List[Point], start: int = 0) -> Tour:
    n = len(points)
    unused = set(range(n))
    tour = [start]
    unused.remove(start)
    curr = start
    while unused:
        nxt = min(unused, key=lambda j: dist(points[curr], points[j]))
        tour.append(nxt)
        unused.remove(nxt)
        curr = nxt
    return tour

# ---------- k-NN per limitar triples (baixa cost) ----------

def k_nearest(points: List[Point], i: int, k: int) -> List[int]:
    n = len(points)
    order = sorted((j for j in range(n) if j != i), key=lambda j: dist(points[i], points[j]))
    return order[:min(k, n-1)]

def build_neighbors(points: List[Point], kNN: int) -> dict:
    return {i: k_nearest(points, i, kNN) for i in range(len(points))}

# ---------- 2-opt bàsic ----------

def delta_cost_2opt(points: List[Point], tour: Tour, i: int, k: int) -> float:
    """
    Canvi 2-opt: invertim el segment (i+1 ... k).
    i < k, posicions en el tour.
    Retorna delta_cost = new - old (negatiu és millora).
    """
    n = len(tour)
    a, b = tour[i], tour[(i+1) % n]
    c, d = tour[k % n], tour[(k+1) % n]

    A, B, C, D = points[a], points[b], points[c], points[d]
    old = dist(A, B) + dist(C, D)
    new = dist(A, C) + dist(B, D)
    return new - old

def apply_2opt(tour: Tour, i: int, k: int) -> Tour:
    """Aplica 2-opt (invertir segment)."""
    n = len(tour)
    if i < k:
        return tour[:i+1] + list(reversed(tour[i+1:k+1])) + tour[k+1:]
    else:
        # envolta (si mai passara), mantenim senzill assumint i < k
        return tour

# ---------- Generació de moviments focalitzats ----------

def generate_moves_around(points: List[Point], tour: Tour, triple: Tuple[int,int,int],
                          X: int, radius: int = 5) -> List[Tuple[int,int]]:
    """
    Proposa fins a ~X moviments 2-opt centrats en posicions de u,v,w.
    radius = quantes posicions al voltant provar per a l’altre extrem.
    """
    u, v, w = triple
    pos = positions_in_tour(tour)
    n = len(tour)

    anchors = [pos[u], pos[v], pos[w]]
    moves = []
    for idx in anchors:
        # Generem candidates (i,k) amb i proper a idx i k en una finestra al voltant
        for di in range(-radius, radius+1):
            i = (idx + di) % n
            j0 = i + 2  # mínim separació per tindre canvi 2-opt consistent
            j1 = i + radius + 2
            for k in range(j0, min(j1, n-1)):
                moves.append((i % n, k % n))
                if len(moves) >= X:
                    return moves
    return moves[:X]

# ---------- Schedule per X ----------

def schedule_X(t: int, X_min: int, X_max: int, stagnation: int) -> int:
    """
    Exemple: creix amb les iteracions o si hi ha estancament,
    però sense passar de X_max.
    """
    inc = t // 2 + stagnation
    return min(X_max, X_min + inc)

# ---------- Algorisme principal ----------

def tsp_local_ME(points: List[Point], T: int = 8, X_min: int = 4, X_max: int = 24,
                 kNN: int = 20, top_triples_per_u: int = 3, radius: int = 6) -> Tour:
    """
    T = rondes macro
    X_min, X_max = dimensió d'exploració (nombre de moviments avaluats per triple)
    kNN = veïnat per limitar triples (quadràtic -> quasi lineal)
    top_triples_per_u = quants triples top usem per cada u
    radius = finestra de posicions per 2-opt focalitzat
    """
    n = len(points)
    tour = initial_greedy_tour(points, start=0)
    best = total_length(points, tour)

    Neigh = build_neighbors(points, kNN)
    stagnation = 0

    for t in range(T):
        X = schedule_X(t, X_min, X_max, stagnation)
        # 1) Construir i ordenar triples (S desc, després D asc)
        triples = []
        for u in range(n):
            cand = []
            Nu = Neigh[u]
            for v in Nu:
                for w in Nu:
                    if v == w: 
                        continue
                    S = saving_score(points, u, v, w)
                    # distància "compacta": dist(u,v)+dist(u,w)
                    D = dist(points[u], points[v]) + dist(points[u], points[w])
                    cand.append((S, D, u, v, w))
            # agafem els top_triples_per_u per u, ordenats
            cand.sort(key=lambda x: (-x[0], x[1]))
            triples.extend(cand[:top_triples_per_u])

        # Ordenació global final (per si vols imposar una prioritat absoluta)
        triples.sort(key=lambda x: (-x[0], x[1]))

        improved = False

        # 2) Per cada triple, genera moviments locals i avalua guany heurístic
        for (S, D, u, v, w) in triples:
            moves = generate_moves_around(points, tour, (u, v, w), X=X, radius=radius)

            best_gain = 0.0
            best_move = None
            for (i, k) in moves:
                dcost = delta_cost_2opt(points, tour, i, k)  # new - old
                # Heurística: volem minimitzar cost -> guany = -dcost * S
                gain = -dcost * S
                if gain > best_gain:
                    best_gain = gain
                    best_move = (i, k)

            if best_move is not None and best_gain > 0:
                # aplica el millor moviment d'esta fornada
                tour = apply_2opt(tour, best_move[0], best_move[1])
                curr = total_length(points, tour)
                if curr + 1e-12 < best:
                    best = curr
                    improved = True

        if improved:
            stagnation = 0
        else:
            stagnation += 1

    return tour

# ---------- Demo ràpida ----------

def demo(seed=7, n=80):
    random.seed(seed)
    points = [(random.random(), random.random()) for _ in range(n)]
    t0 = initial_greedy_tour(points)
    L0 = total_length(points, t0)

    tour = tsp_local_ME(points, T=8, X_min=4, X_max=20, kNN=15, top_triples_per_u=2, radius=5)
    L1 = total_length(points, tour)

    print(f"Longitud inicial (greedy): {L0:.5f}")
    print(f"Longitud final (heurística ME local): {L1:.5f}")
    print(f"Millora: {L0 - L1:.5f}")

if __name__ == "__main__":
    demo()

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
ME locals (quadràtica/cúbica) i test de NO-unificació
Autor: VMA + Copilot
Ús: python me_local_test.py

Què fa:
- Construeix finestres locals al voltant de p^2 (quadràtic) i p^3 (cúbic)
- Fa bootstrap + SVD per detectar col·lapse local (ME local)
- Comprova el col·lapse global (totes les finestres juntes)
- Si locals=Sí i global=No -> NO-UNIFICACIÓ al teu model

Com ampliar:
- Ajusta les llistes PRIMES_SQUARE i PRIMES_CUBE (per ex. afegeix 13,17,19)
- Ajusta EPS_SQ i EPS_CB si vols més/menys tolerància al log
- Ajusta TAU/CONSENSUS/BOOTSTRAP_N segons la finor del test
"""
from dataclasses import dataclass, field
from typing import List, Optional, Dict, Tuple
import numpy as np
from math import log, exp

# ---------- Dades d'entrada (edita ací) ----------
PRIMES_SQUARE = [5, 7, 11]          # finestra quadràtica: 5^2, 7^2, 11^2
PRIMES_CUBE   = [5, 7, 11]          # finestra cúbica:    5^3, 7^3, 11^3
# Exemple d’ampliació (descomenta si vols):
# PRIMES_CUBE   = [5, 7, 11, 13, 17, 19]

EPS_SQ = 0.15   # tolerància (±) en log(E) per a quadrats
EPS_CB = 0.18   # tolerància (±) en log(E) per a cubes

# Paràmetres del test (pots ajustar-los)
TAU          = 5e-3   # llindar col·lapse: sigma_min/sigma_max < TAU => degenerat
CONSENSUS    = 0.90   # % de bootstraps necessaris per declarar ME local/global
BOOTSTRAP_N  = 900    # mostres bootstrap
SEED         = 2026   # llavor RNG


# ---------- Motor de test ----------
@dataclass
class LogObs:
    y: float       # valor central (log(E))
    eps: float     # tolerància ± en log(E)
    tag: str = ""  # etiqueta opcional

@dataclass
class MEQTesterV3:
    tau: float = TAU
    consensus: float = CONSENSUS
    bootstrap_N: int = BOOTSTRAP_N
    seed: int = SEED
    obs: List[LogObs] = field(default_factory=list)

    def add_log_obs(self, y: float, eps: float, tag: str = ""):
        self.obs.append(LogObs(y=y, eps=eps, tag=tag))

    @staticmethod
    def _near_k_power_residual(E: float, k: int) -> float:
        """Distància a la k-èsima potència perfecta més propera (en l’espai real)."""
        x = E ** (1.0 / k)
        r = round(x)
        return abs(x - r)

    def _features_from_E(self, E: float) -> np.ndarray:
        """
        Features per distingir règims quadràtic vs cúbic:
        - f1: log(E)
        - f2: log(1 + distància a quadrat perfecte)
        - f3: log(1 + distància a cub perfecte)
        - f4: log(1 + distància a 5a potència perfecta)
        - f5: contrast entre distàncies a quadrat/cub (qualsevol règim dominant)
        """
        y = log(max(E, 1e-300))
        d2 = self._near_k_power_residual(E, 2)
        d3 = self._near_k_power_residual(E, 3)
        d5 = self._near_k_power_residual(E, 5)
        f1 = y
        f2 = np.log(1.0 + d2)
        f3 = np.log(1.0 + d3)
        f4 = np.log(1.0 + d5)
        mn = min(d2, d3) + 1e-12
        mx = max(d2, d3) + 1e-12
        f5 = np.log(mx / mn)
        return np.array([f1, f2, f3, f4, f5], dtype=float)

    def _sample_matrix(self, rng: np.random.Generator, indices: Optional[List[int]] = None) -> np.ndarray:
        rows = []
        src = self.obs if indices is None else [self.obs[i] for i in indices]
        for ob in src:
            y_s = rng.uniform(ob.y - ob.eps, ob.y + ob.eps)  # bootstrapping dins de la tolerància
            E_s = exp(y_s)
            rows.append(self._features_from_E(E_s))
        Y = np.vstack(rows)
        # centrat de columnes (treiem mitjana): provem col·lapse afí
        return Y - Y.mean(axis=0, keepdims=True)

    def _degenerate_fraction(self, indices: Optional[List[int]] = None) -> float:
        rng = np.random.default_rng(self.seed)
        deg = 0
        for _ in range(self.bootstrap_N):
            Yc = self._sample_matrix(rng, indices)
            if Yc.shape[0] < 3 or Yc.shape[1] < 2:
                continue
            S = np.linalg.svd(Yc, full_matrices=False, compute_uv=False)
            if S[0] <= 0:
                continue
            if (S[-1] / S[0]) < self.tau:
                deg += 1
        return deg / max(1, self.bootstrap_N)

    def local_tests(self, window_indices: List[List[int]]) -> List[Dict]:
        out = []
        for idx in window_indices:
            df = self._degenerate_fraction(indices=idx)
            out.append({
                "indices": idx,
                "degenerate_fraction": df,
                "local_ME_exists": df >= self.consensus
            })
        return out

    def global_unification_test(self) -> Dict:
        df = self._degenerate_fraction(indices=None)
        return {"degenerate_fraction_all": df, "global_ME_exists": df >= self.consensus}


# ---------- Utilitats ----------
def build_power_logs(primes: List[int], k: int, eps: float, label: str) -> List[Tuple[float, float, str]]:
    """
    Construeix (log(E), eps, tag) per a E = p^k sobre primes donades.
    label s’afegeix al tag per separar finestres (p.ex. 'SQ' o 'CB').
    """
    out = []
    for p in primes:
        E = float(p ** k)
        out.append((log(E), eps, f"{int(E)}={p}^{k} [{label}]"))
    return out

def print_human_summary(title: str, local_res: List[Dict], global_res: Dict):
    print("\n" + "="*70)
    print(f"{title}")
    print("="*70)
    for i, r in enumerate(local_res, 1):
        idx = r["indices"]
        print(f"- Finestra {i} (índexos {idx}): "
              f"deg_frac={r['degenerate_fraction']:.3f}  "
              f"ME_local={r['local_ME_exists']}")
    print(f"Global: deg_frac_all={global_res['degenerate_fraction_all']:.3f}  "
          f"ME_global={global_res['global_ME_exists']}")
    # Decisió de NO-unificació (locals sí, global no):
    k = sum(1 for r in local_res if r["local_ME_exists"])
    non_unif = (k > 0) and (not global_res["global_ME_exists"])
    print(f"NO-UNIFICACIÓ (locals sí i global no)?  {non_unif}")
    print("="*70)


# ---------- Demo principal ----------
def main():
    tester = MEQTesterV3()

    # 1) Afegim finestra QUADRÀTICA
    sq_obs = build_power_logs(PRIMES_SQUARE, k=2, eps=EPS_SQ, label="SQ")
    for y, eps, tag in sq_obs:
        tester.add_log_obs(y, eps, tag)

    # 2) Afegim finestra CÚBICA

    offset = len(sq_obs)
    for y, eps, tag in cb_obs:

    # Índexos de finestra (locals)
    sq_idx = list(range(0, len(sq_obs)))                  # [0..len(sq)-1]
    cb_idx = list(range(offset, offset + len(cb_obs)))    # [offset..offset+len(cb)-1]

    # 3) Tests locals
    local_results = tester.local_tests([sq_idx, cb_idx])

    # 4) Test global (totes les files)
    global_result = tester.global_unification_test()

    # 5) Resum humà
    print_human_summary("ME locals quadràtica i cúbica", local_results, global_result)

    # 6) També escopim JSON estructurat (per si vols parsejar)
    import json
    out = {
        "params": {
            "tau": tester.tau,
            "consensus": tester.consensus,
            "bootstrap_N": tester.bootstrap_N,
            "seed": tester.seed,
            "eps_sq": EPS_SQ,
            "eps_cb": EPS_CB,
            "primes_square": PRIMES_SQUARE,
            "primes_cube": PRIMES_CUBE
        },
        "local": local_results,
        "global": global_result,
        "non_unification": (sum(1 for r in local_results if r["local_ME_exists"]) > 0) and (not global_result["global_ME_exists"])
    }
    print("\nJSON_RESULT =")

if __name__ == "__main__":

import numpy as np
from math import exp, log, sqrt
from dataclasses import dataclass, field
from typing import List, Optional, Dict

@dataclass
class LogObs:
    y: float           # valor central de log(E)
    eps: float         # tolerància ± en log(E)
    tag: str = ""

@dataclass
class MEQTesterV2:
    tau: float = 1e-3          # llindar de col·lapse: sigma_min/sigma_max < tau
    consensus: float = 0.90    # % de bootstraps “degenerats” per declarar ME local/global
    bootstrap_N: int = 600
    seed: int = 2026
    obs: List[LogObs] = field(default_factory=list)

    def add_log_obs(self, y: float, eps: float, tag: str = ""):
        self.obs.append(LogObs(y=y, eps=eps, tag=tag))

    def _features_from_E(self, E: float) -> np.ndarray:
        # Features no trivials (evitem relacions exactes): totes funcionals de E
        y = log(max(E, 1e-300))
        f1 = y
        f2 = log(1.0 + sqrt(E))
        f3 = log(1.0 + E**(1/3))
        f4 = log(1.0 + E**(1/5))
        return np.array([f1, f2, f3, f4], dtype=float)

    def _sample_matrix(self, rng: np.random.Generator, indices: Optional[List[int]] = None) -> np.ndarray:
        rows = []
        src = self.obs if indices is None else [self.obs[i] for i in indices]
        for ob in src:
            y_s = rng.uniform(ob.y - ob.eps, ob.y + ob.eps)
            E_s = exp(y_s)
            rows.append(self._features_from_E(E_s))
        Y = np.vstack(rows)
        # Centrat de columnes (treiem mitjana): prova de col·lapse afí
        Yc = Y - Y.mean(axis=0, keepdims=True)
        return Yc

    def _degenerate_fraction(self, indices: Optional[List[int]] = None) -> float:
        rng = np.random.default_rng(self.seed)
        deg = 0
        for _ in range(self.bootstrap_N):
            Yc = self._sample_matrix(rng, indices)
            if Yc.shape[0] < 3 or Yc.shape[1] < 2:
                continue
            S = np.linalg.svd(Yc, full_matrices=False, compute_uv=False)
            if S[0] <= 0: 
                continue
            if (S[-1] / S[0]) < self.tau:
                deg += 1
        return deg / max(1, self.bootstrap_N)

    def local_tests(self, window: int = 3) -> List[Dict]:
        n = len(self.obs)
        out = []
        if n < window: return out
        for s in range(0, n - window + 1):
            idx = list(range(s, s + window))
            df = self._degenerate_fraction(indices=idx)
            out.append({ "indices": idx, "degenerate_fraction": df, "local_ME_exists": df >= self.consensus })
        return out

    def global_unification_test(self) -> Dict:
        df = self._degenerate_fraction(indices=None)
        return {"degenerate_fraction_all": df, "global_ME_exists": df >= self.consensus}

    def decision(self, window: int = 3) -> Dict:
        L = self.local_tests(window)
        G = self.global_unification_test()
        k = sum(1 for r in L if r["local_ME_exists"]) if L else 0
        return { "local_windows": L, "global": G, "num_local_ME": k,
                 "non_unification": (k > 0) and (not G["global_ME_exists"]) }

def pca_rank_r_reconstruction_error(Yc: np.ndarray, r: int=1) -> float:
    # Yc: dades centrades (x × d). Retorna RMSE normalitzat per norma Frobenius.
    U, S, Vt = np.linalg.svd(Yc, full_matrices=False)
    # Retalla a rang r
    Sr = np.zeros_like(S)
    Sr[:r] = S[:r]
    Y_hat = (U * Sr) @ Vt
    num = np.linalg.norm(Yc - Y_hat, 'fro')
    den = np.linalg.norm(Yc, 'fro') + 1e-15
    return float(num/den)

"""
ME locals (quadràtica/cúbica) i test de NO-unificació
Autor: VMA + Copilot
Ús: python me_local_test.py

Què fa:
- Construeix finestres locals al voltant de p^2 (quadràtic) i p^3 (cúbic)
- Fa bootstrap + SVD per detectar col·lapse local (ME local)
- Comprova el col·lapse global (totes les finestres juntes)
- Si locals=Sí i global=No -> NO-UNIFICACIÓ al teu model

Com ampliar:
- Ajusta les llistes PRIMES_SQUARE i PRIMES_CUBE (per ex. afegeix 13,17,19)
- Ajusta EPS_SQ i EPS_CB si vols més/menys tolerància al log
- Ajusta TAU/CONSENSUS/BOOTSTRAP_N segons la finor del test
"""
from dataclasses import dataclass, field
from typing import List, Optional, Dict, Tuple
import numpy as np
from math import log, exp

# ---------- Dades d'entrada (edita ací) ----------
PRIMES_SQUARE = [5, 7, 11]          # finestra quadràtica: 5^2, 7^2, 11^2
PRIMES_CUBE   = [5, 7, 11]          # finestra cúbica:    5^3, 7^3, 11^3
# Exemple d’ampliació (descomenta si vols):
# PRIMES_CUBE   = [5, 7, 11, 13, 17, 19]

EPS_SQ = 0.15   # tolerància (±) en log(E) per a quadrats
EPS_CB = 0.18   # tolerància (±) en log(E) per a cubes

# Paràmetres del test (pots ajustar-los)
TAU          = 5e-3   # llindar col·lapse: sigma_min/sigma_max < TAU => degenerat
CONSENSUS    = 0.90   # % de bootstraps necessaris per declarar ME local/global
BOOTSTRAP_N  = 900    # mostres bootstrap
SEED         = 2026   # llavor RNG


# ---------- Motor de test ----------
@dataclass
class LogObs:
    y: float       # valor central (log(E))
    eps: float     # tolerància ± en log(E)
    tag: str = ""  # etiqueta opcional

@dataclass
class MEQTesterV3:
    tau: float = TAU
    consensus: float = CONSENSUS
    bootstrap_N: int = BOOTSTRAP_N
    seed: int = SEED
    obs: List[LogObs] = field(default_factory=list)

    def add_log_obs(self, y: float, eps: float, tag: str = ""):
        self.obs.append(LogObs(y=y, eps=eps, tag=tag))

    @staticmethod
    def _near_k_power_residual(E: float, k: int) -> float:
        """Distància a la k-èsima potència perfecta més propera (en l’espai real)."""
        x = E ** (1.0 / k)
        r = round(x)
        return abs(x - r)

    def _features_from_E(self, E: float) -> np.ndarray:
        """
        Features per distingir règims quadràtic vs cúbic:
        - f1: log(E)
        - f2: log(1 + distància a quadrat perfecte)
        - f3: log(1 + distància a cub perfecte)
        - f4: log(1 + distància a 5a potència perfecta)
        - f5: contrast entre distàncies a quadrat/cub (qualsevol règim dominant)
        """
        y = log(max(E, 1e-300))
        d2 = self._near_k_power_residual(E, 2)
        d3 = self._near_k_power_residual(E, 3)
        d5 = self._near_k_power_residual(E, 5)
        f1 = y
        f2 = np.log(1.0 + d2)
        f3 = np.log(1.0 + d3)
        f4 = np.log(1.0 + d5)
        mn = min(d2, d3) + 1e-12
        mx = max(d2, d3) + 1e-12
        f5 = np.log(mx / mn)
        return np.array([f1, f2, f3, f4, f5], dtype=float)

    def _sample_matrix(self, rng: np.random.Generator, indices: Optional[List[int]] = None) -> np.ndarray:
        rows = []
        src = self.obs if indices is None else [self.obs[i] for i in indices]
        for ob in src:
            y_s = rng.uniform(ob.y - ob.eps, ob.y + ob.eps)  # bootstrapping dins de la tolerància
            E_s = exp(y_s)
            rows.append(self._features_from_E(E_s))
        Y = np.vstack(rows)
        # centrat de columnes (treiem mitjana): provem col·lapse afí
        return Y - Y.mean(axis=0, keepdims=True)

    def _degenerate_fraction(self, indices: Optional[List[int]] = None) -> float:
        rng = np.random.default_rng(self.seed)
        deg = 0
        for _ in range(self.bootstrap_N):
            Yc = self._sample_matrix(rng, indices)
            if Yc.shape[0] < 3 or Yc.shape[1] < 2:
                continue
            S = np.linalg.svd(Yc, full_matrices=False, compute_uv=False)
            if S[0] <= 0:
                continue
            if (S[-1] / S[0]) < self.tau:
                deg += 1
        return deg / max(1, self.bootstrap_N)

    def local_tests(self, window_indices: List[List[int]]) -> List[Dict]:
        out = []
        for idx in window_indices:
            df = self._degenerate_fraction(indices=idx)
            out.append({
                "indices": idx,
                "degenerate_fraction": df,
                "local_ME_exists": df >= self.consensus
            })
        return out

    def global_unification_test(self) -> Dict:
        df = self._degenerate_fraction(indices=None)
        return {"degenerate_fraction_all": df, "global_ME_exists": df >= self.consensus}


# ---------- Utilitats ----------
def build_power_logs(primes: List[int], k: int, eps: float, label: str) -> List[Tuple[float, float, str]]:
    """
    Construeix (log(E), eps, tag) per a E = p^k sobre primes donades.
    label s’afegeix al tag per separar finestres (p.ex. 'SQ' o 'CB').
    """
    out = []
    for p in primes:
        E = float(p ** k)
        out.append((log(E), eps, f"{int(E)}={p}^{k} [{label}]"))
    return out

def print_human_summary(title: str, local_res: List[Dict], global_res: Dict):
    print("\n" + "="*70)
    print(f"{title}")
    print("="*70)
    for i, r in enumerate(local_res, 1):
        idx = r["indices"]
        print(f"- Finestra {i} (índexos {idx}): "
              f"deg_frac={r['degenerate_fraction']:.3f}  "
              f"ME_local={r['local_ME_exists']}")
    print(f"Global: deg_frac_all={global_res['degenerate_fraction_all']:.3f}  "
          f"ME_global={global_res['global_ME_exists']}")
    # Decisió de NO-unificació (locals sí, global no):
    k = sum(1 for r in local_res if r["local_ME_exists"])
    non_unif = (k > 0) and (not global_res["global_ME_exists"])
    print(f"NO-UNIFICACIÓ (locals sí i global no)?  {non_unif}")
    print("="*70)

"""
Heurística geomètrica per al TSP amb ME locals i dimensió X
-----------------------------------------------------------

- Score d'estalvi per triples (u,v,w)
- Ordenació per score desc i distància asc
- Exploració controlada per X (dimensió)
- 2-opt focalitzat al voltant del triple
- Selecció del millor guany heurístic

Autor: VMA + Copilot (2026-01-24)
"""

import math
import random
from collections import defaultdict
from typing import List, Tuple

Point = Tuple[float, float]
Tour = List[int]

# ---------- Distàncies i utilitats ----------

def dist(p: Point, q: Point) -> float:
    return math.hypot(p[0]-q[0], p[1]-q[1])

def total_length(points: List[Point], tour: Tour) -> float:
    L = 0.0
    n = len(tour)
    for i in range(n):
        a = points[tour[i]]
        b = points[tour[(i+1) % n]]
        L += dist(a, b)
    return L

def positions_in_tour(tour: Tour) -> dict:
    """Mapeig node -> posició al tour (O(n))."""
    return {node: i for i, node in enumerate(tour)}

# ---------- Score d'estalvi per triples ----------

def saving_score(points: List[Point], u: int, v: int, w: int, eps: float = 1e-9) -> float:
    """
    Interpretació de la teua fórmula:
    S(u,v,w) = ((d(u,w)+d(u,v))^2 / (d(u,v)+eps)) * d(u,w)
    Després normalitzem lleugerament per evitar dominàncies exagerades.
    """
    duv = dist(points[u], points[v])
    duw = dist(points[u], points[w])
    raw = ((duw + duv) ** 2) / (duv + eps) * duw
    # Normalització suau per no sobrepremiar distàncies molt grans:
    norm = raw / (duw + duv + 1e-9)
    return norm

# ---------- Inicialització (tour greedy) ----------

def initial_greedy_tour(points: List[Point], start: int = 0) -> Tour:
    n = len(points)
    unused = set(range(n))
    tour = [start]
    unused.remove(start)
    curr = start
    while unused:
        nxt = min(unused, key=lambda j: dist(points[curr], points[j]))
        tour.append(nxt)
        unused.remove(nxt)
        curr = nxt
    return tour

# ---------- k-NN per limitar triples (baixa cost) ----------

def k_nearest(points: List[Point], i: int, k: int) -> List[int]:
    n = len(points)
    order = sorted((j for j in range(n) if j != i), key=lambda j: dist(points[i], points[j]))
    return order[:min(k, n-1)]

def build_neighbors(points: List[Point], kNN: int) -> dict:
    return {i: k_nearest(points, i, kNN) for i in range(len(points))}

# ---------- 2-opt bàsic ----------

def delta_cost_2opt(points: List[Point], tour: Tour, i: int, k: int) -> float:
    """
    Canvi 2-opt: invertim el segment (i+1 ... k).
    i < k, posicions en el tour.
    Retorna delta_cost = new - old (negatiu és millora).
    """
    n = len(tour)
    a, b = tour[i], tour[(i+1) % n]
    c, d = tour[k % n], tour[(k+1) % n]

    A, B, C, D = points[a], points[b], points[c], points[d]
    old = dist(A, B) + dist(C, D)
    new = dist(A, C) + dist(B, D)
    return new - old

def apply_2opt(tour: Tour, i: int, k: int) -> Tour:
    """Aplica 2-opt (invertir segment)."""
    n = len(tour)
    if i < k:
        return tour[:i+1] + list(reversed(tour[i+1:k+1])) + tour[k+1:]
    else:
        # envolta (si mai passara), mantenim senzill assumint i < k
        return tour

# ---------- Generació de moviments focalitzats ----------

def generate_moves_around(points: List[Point], tour: Tour, triple: Tuple[int,int,int],
                          X: int, radius: int = 5) -> List[Tuple[int,int]]:
    """
    Proposa fins a ~X moviments 2-opt centrats en posicions de u,v,w.
    radius = quantes posicions al voltant provar per a l’altre extrem.
    """
    u, v, w = triple
    pos = positions_in_tour(tour)
    n = len(tour)

    anchors = [pos[u], pos[v], pos[w]]
    moves = []
    for idx in anchors:
        # Generem candidates (i,k) amb i proper a idx i k en una finestra al voltant
        for di in range(-radius, radius+1):
            i = (idx + di) % n
            j0 = i + 2  # mínim separació per tindre canvi 2-opt consistent
            j1 = i + radius + 2
            for k in range(j0, min(j1, n-1)):
                moves.append((i % n, k % n))
                if len(moves) >= X:
                    return moves
    return moves[:X]

# ---------- Schedule per X ----------

def schedule_X(t: int, X_min: int, X_max: int, stagnation: int) -> int:
    """
    Exemple: creix amb les iteracions o si hi ha estancament,
    però sense passar de X_max.
    """
    inc = t // 2 + stagnation
    return min(X_max, X_min + inc)

# ---------- Algorisme principal ----------

def tsp_local_ME(points: List[Point], T: int = 8, X_min: int = 4, X_max: int = 24,
                 kNN: int = 20, top_triples_per_u: int = 3, radius: int = 6) -> Tour:
    """
    T = rondes macro
    X_min, X_max = dimensió d'exploració (nombre de moviments avaluats per triple)
    kNN = veïnat per limitar triples (quadràtic -> quasi lineal)
    top_triples_per_u = quants triples top usem per cada u
    radius = finestra de posicions per 2-opt focalitzat
    """
    n = len(points)
    tour = initial_greedy_tour(points, start=0)
    best = total_length(points, tour)

    Neigh = build_neighbors(points, kNN)
    stagnation = 0

    for t in range(T):
        X = schedule_X(t, X_min, X_max, stagnation)
        # 1) Construir i ordenar triples (S desc, després D asc)
        triples = []
        for u in range(n):
            cand = []
            Nu = Neigh[u]
            for v in Nu:
                for w in Nu:
                    if v == w: 
                        continue
                    S = saving_score(points, u, v, w)
                    # distància "compacta": dist(u,v)+dist(u,w)
                    D = dist(points[u], points[v]) + dist(points[u], points[w])
                    cand.append((S, D, u, v, w))
            # agafem els top_triples_per_u per u, ordenats
            cand.sort(key=lambda x: (-x[0], x[1]))
            triples.extend(cand[:top_triples_per_u])

        # Ordenació global final (per si vols imposar una prioritat absoluta)
        triples.sort(key=lambda x: (-x[0], x[1]))

        improved = False

        # 2) Per cada triple, genera moviments locals i avalua guany heurístic
        for (S, D, u, v, w) in triples:
            moves = generate_moves_around(points, tour, (u, v, w), X=X, radius=radius)

            best_gain = 0.0
            best_move = None
            for (i, k) in moves:
                dcost = delta_cost_2opt(points, tour, i, k)  # new - old
                # Heurística: volem minimitzar cost -> guany = -dcost * S
                gain = -dcost * S
                if gain > best_gain:
                    best_gain = gain
                    best_move = (i, k)

            if best_move is not None and best_gain > 0:
                # aplica el millor moviment d'esta fornada
                tour = apply_2opt(tour, best_move[0], best_move[1])
                curr = total_length(points, tour)
                if curr + 1e-12 < best:
                    best = curr
                    improved = True

        if improved:
            stagnation = 0
        else:
            stagnation += 1

    return tour
